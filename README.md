# forked-assets

Your GitSort capability warehouse — the permanent, sovereign home of every
capability you approve. GitSort owns nothing; if it disappears, this repo
(and everything in it) remains yours.

- `syntheses/`, `capabilities/` — the assets themselves
- `receipts/` — the audit for each asset (sources, what was merged/removed, confidence, lineage)
- `manifests/assets.json` — the index GitSort reads to rebuild your catalog
